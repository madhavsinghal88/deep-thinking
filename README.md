# Deep Thinking Mode

A minimalist, distraction-free thinking canvas for programmers solving difficult problems like LeetCode or system design.

Inspired by IIT/IIM topper productivity environments.

## Philosophy

- Extreme minimalism
- Zero distraction
- Cognitive clarity
- Monk-level focus

## Features

- **Dark/Light Mode** - Matte black (#0d0d0d) or warm off-white background
- **Centered Thinking Area** - Large editable text box with auto-save (localStorage)
- **Pomodoro Timer** - 25 or 50 minute presets
- **Digital Clock** - Subtle, minimal
- **Ambient Sound** - Brown noise generator (rain-like)
- **Rotating Quotes** - Motivational, changes every minute
- **Mobile Responsive** - Works on iPhone and laptop
- **No Frameworks** - Pure HTML + CSS + Vanilla JS

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Z` | Zen Mode (hidden) |
| `ESC` | Clear screen / Exit Zen |
| `Ctrl+S` | Save notes |
| `F` | Toggle fullscreen |

## Zen Mode

Press `Z` to enter Zen Mode:
- Entire screen becomes blank
- No UI visible
- Only blinking cursor
- Pure thinking canvas

Press `ESC` to exit.

## Distraction Blocking

- Right-click disabled
- No scrolling
- No animations except subtle fades

## Usage

Open `deep-thinking.html` in your browser.

Or serve locally:

```bash
python3 -m http.server 5007
# Open http://localhost:5007/deep-thinking.html
```
